# Search Engine Using Google API

Hi, inspired from Clever Programmers, I create google clone with working search engine using google search API, another technology used is :

- React (CRA)
- React-Router-Dom
- @material-ui/core
- @material-ui/icons
- React Context API
- Google Custom REST API
